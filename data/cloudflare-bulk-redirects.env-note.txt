Bulk Redirects CSV is not generated until you set a full origin for the NEW site.

Example:
  MIGRATION_TARGET_ORIGIN=https://nena.pages.dev npm run redirects:legacy

Optional:
  LEGACY_REDIRECT_SOURCE_HOST=www.nenabozeman.org   (default)
  LEGACY_ALSO_APEX=1   (duplicate each row for nenabozeman.org without www)

CSV rules (no header row):
https://developers.cloudflare.com/rules/url-forwarding/bulk-redirects/reference/csv-file-format/
